What is DOM and what is the purpose of DOM?

DOM is an application programming interface, dom means, Document Object Model.Dom is like tree because of its nodes by using dom we can create, change, or remove elements from a website document more efficiently.

How will you select HTML elements using DOM? Name the DOM methods?

We can select HTML elements using Dom in different ways. Those methods are:


> document.getElementsByTagName()
> document.getElementsByClassName()
> document.getElementById()
> document.querySelector()
> document.querySelectorAll()

What is event bubble?

In HTML we can nest tags and have parent elements that can have multiple child elements.If we call a child it will check its parent node, at each stage it will check the parent node. Which feels like a bubble.

What is a callback function and why will you use it?

A callback function is a function that is passed as a parameter to another function and called when a event happens
